HOW TO USE

It's easier than it looks!

Make sure all options are input in lowercase, and without any spaces. Any capital letters will not work.

Beginning of include

Item # of SCP

Clearance Level USE DIGITS (5,4,3,2,1,0) (See guide here)

Container Class (Also known as Object Class)

OPTIONAL Secondary Class (If Using, set Container Class to Esoteric)

If secondary class is used, place URL to Class Icon here

Disruption Class (See Guide & Read Below for Explanation)

Risk Class (See Guide & Read Below for Explanation)

End of include

[[include :scp-wiki:component:anomaly-class-bar-source

|item-number= XXXX

|clearance= #

|secondary-class= none

|secondary-icon= https://urlhere.com

|disruption-class= vlam

|risk-class= warning

]]
